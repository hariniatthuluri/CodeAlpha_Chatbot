# 🤖 Python Chatbot | CodeAlpha Internship

A simple rule-based chatbot built using Python that interacts with users through predefined responses and basic logic.

---

## 🚀 Features
- Interactive command-line chatbot  
- Handles multiple user inputs  
- Remembers user name during the session  
- Uses randomized responses for natural interaction  
- Clean and beginner-friendly implementation  

---

## 🧠 Concepts Used
- Loops (`while`)  
- Conditional Statements (`if-elif-else`)  
- String Handling  
- Lists & Random Module  
- User Input Processing
